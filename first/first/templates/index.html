Abstract

For our project we created an application that scrapes tweets in real-time across the continental United States. Our application then creates and displays dynamic map of the United States showing the location and sentiment of the streamed tweets. To do the sentiment analysis we used the TextBlob python library, and to create the map we used the gmplot library. Our application is built using a Flask front end which allows the user to specify their various desired inputs, such as a location to center the map on and a keyword to parse the database with. The output from our application will be a rendered map depicting the overall emotion of the United States, centered on the specified location or showing only tweets containing the specified keyword. Additionally, this map displays the most common words in both positive and negative sentiment tweets as a 'event discovery' method, however, this functionality is pretty basic at this point in time.
