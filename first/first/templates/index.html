Why sentiment analysis?

Sentiment analysis of data has always been an important, but has become a harder but interesting task as applications like Twitter have grown. Tweets contain data that can be short or informal, which can be a problem, but more importantly it contains a large dataset which can help with testing and analyzing trends. Understanding the mood people show off to different phenomenon's can be used in many different ways to adjust the phoenomenon.

Objective

Our goal was to design a tweet sentiment analyzing application. The application would preform a series of tasks on twitter data we access using Tweepy. The application has the ability of collecting tweets and storing them for future analysis. The data can be displayed to the user as raw data, or as a map that displays the results of the sentiment analysis algorithm.
